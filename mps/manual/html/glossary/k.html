keyword argument

An optional argument to a function call, identified by an associated keyword.

In the MPS

Keyword arguments are passed to functions in the MPS interface as arrays of structures of type mps_arg_s. See Keyword arguments.

kilobyte

Also known as

kB.

A kilobyte is 1024 bytes(1).

See byte(1) for general information on this and related quantities.

The standard abbreviation is “kB”, but “KB” is often used by people unfamiliar with the metric system.
