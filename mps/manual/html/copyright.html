Memory Pool System Kit Open Source License¶

This is the license under which the Memory Pool System Kit is available as open source.

It is not the only way the MPS is licensed.

If the licensing terms aren’t suitable for you (for example, you’re developing a closed-source commercial product or a compiler run-time system) you can easily license the MPS under different terms from Ravenbrook. Please write to us at mailto:mps-questions@ravenbrook.com for more information.

The open source license for the MPS is the Sleepycat License also known as the “Berkeley Database License”. This license is GPL compatible and OSI approved. The MPS is “multi licensed” in a manner similar to MySQL.
