Purpose

The quote document type is for documents which are extracts from sources outside the project, for example, parts of papers or stuff copied from books.

Content

The content of a quote document is the text copied from the original source.

Tagging

Quoted material should be tagged with the tag of its source (a document of type bib ), followed by a name for the quotation. For example, "book.gilb93.rule.generic".
