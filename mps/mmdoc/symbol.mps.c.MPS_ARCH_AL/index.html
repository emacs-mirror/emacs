Description of the MPS C Interface Symbol "MPS_ARCH_AL"

obsolete

GRM , 1997-05-02

Name

MPS_ARCH_AL

Summary

"MPS_ARCH_AL" is a C preprocessor macro that indicates, if defined, that the target processor architecture of the compilation is a member of the DEC Alpha family. It is defined, if appropriate, by "mpstd.h".

Associated Protocols

None.

Resources

mpstd.h.

Description

See summary.

Example

#ifdef MPS_ARCH_AL

typedef struct RegisterFile {
unsigned long v0;
unsigned long t0, t1, t2, t3, t4, t5, t6, t7; unsigned long s0, s1, s2, s3, s4, s5;
unsigned long fp;
unsigned long a0, a1, a2, a3, a4, a5; unsigned long t8, t9, t10, t11;
unsigned long ra;
unsigned long t12;
unsigned long at, gp, sp, zero; unsigned long fir;
unsigned psr;
} RegisterFile;

#endif /* MPS_ARCH_AL */

Error Handling

Not applicable.

See Also

MPS_PF_*, MPS_OS_*, MPS_BUILD_*, MPS_ARCH_*

Notes

Internal Notes

I'm not sure that the user ought to be using these symbols. GavinM 1997-05-01
