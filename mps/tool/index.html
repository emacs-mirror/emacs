Memory Pool System Project

Memory Pool System Product Tools

Richard Brooksby, Ravenbrook Limited, 2002-06-18

1. Introduction

This document lists the tools which have been written to develop the Memory Pool System product.

This document will be updated as new tools are created.

The readership of this document is anyone developing or extending the product sources.

This document is not confidential.

2. Tools

  • The Python script gcovfmt.py formats the output of the gcov coverage tool into a summary table. It runs only on OS X, where it is invoked from the Xcode project after running the test suite for the “Debug” configuration.

  • The shell script testrun.sh implements the “testrun” make target. On FreeBSD and Linux it is invoked from comm.gmk and on OS X it is invoked from the Xcode project.

  • The Python script test-runner.py builds and runs tests. As of 2013-05-24 it is not used.
