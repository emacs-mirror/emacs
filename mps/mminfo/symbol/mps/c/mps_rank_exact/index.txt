NAME

mps_rank_exact

SUMMARY

Used to declare references which the client wishes to be exact references.

ASSOCIATED PROTOCOLS

Object Allocation, protocol.reference

SYNTAX

mps_rank_exact()

TYPE

mps_rank_t mps_rank_exact(void);

ARGUMENTS

No arguments.

INITIAL/DEFAULT VALUES

NA

RETURNED VALUES

Returns a rank (see mps_rank_t) which can be used to declare references to be 
exact references.
