NAME

mps_message_queue_type


SUMMARY

If there are any messages on the queue of the specified arena, then this 
function returns "true", and also returns the type of the first message via 
"message_type_return".  Otherwise it returns "false".


ASSOCIATED PROTOCOLS

Message.


SYNTAX

mps_bool_t mps_message_queue_type(mps_message_type_t *message_type_return, 
mps_arena_t arena)


ARGUMENTS

message_type_return -- the type of the first message on the queue of the 
specified arena
arena -- the arena 


INITIAL/DEFAULT VALUES

Not applicable.


RETURNED VALUES

"True" if there are any messages on the queue of the specified arena, "false" 
if not.

Not applicable.
