NAME

mps_arena_class_t


SUMMARY

"mps_arena_class_t" is the type of arena classes.  It is opaque.


ASSOCIATED PROTOCOLS

Arena.


TYPE

mps_arena_class_t arena;

"mps_arena_class_s" is an incomplete structure type used only to declare the 
opaque type "mps_arena_class_t".


RESOURCES

mps.h


DESCRIPTION

None.


EXAMPLE

The definition of the client arena class in the "mpsacl.h" header:

extern mps_arena_class_t mps_arena_class_cl(void);
