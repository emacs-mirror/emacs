NAME

MPS_ARCH_AL


SUMMARY

"MPS_ARCH_AL" is a C preprocessor macro that indicates, if defined, that the 
target processor architecture of the compilation is a member of the DEC Alpha 
family.  It is defined, if appropriate, by "mpstd.h".


ASSOCIATED PROTOCOLS

None.


RESOURCES

mpstd.h.


DESCRIPTION

See summary.


EXAMPLE

#ifdef MPS_ARCH_AL
typedef struct RegisterFile {
  unsigned long v0;
  unsigned long t0, t1, t2, t3, t4, t5, t6, t7;
  unsigned long s0, s1, s2, s3, s4, s5;
  unsigned long fp;
  unsigned long a0, a1, a2, a3, a4, a5;
  unsigned long t8, t9, t10, t11;
  unsigned long ra;
  unsigned long t12;
  unsigned long at, gp, sp, zero;
  unsigned long fir;
  unsigned psr;
} RegisterFile;
#endif /* MPS_ARCH_AL */


ERROR HANDLING

Not applicable.
