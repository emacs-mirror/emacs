NAME

mps_message_type_finalization


SUMMARY

"mps_message_type_finalization" returns the type of finalization messages.


ASSOCIATED PROTOCOLS

Message, finalization.


SYNTAX

mps_message_type_t mps_message_type_finalization()


ARGUMENTS


INITIAL/DEFAULT VALUES

Not applicable.


RETURNED VALUES

The type of finalization messages.


RESOURCES

Not applicable.


DESCRIPTION

"mps_message_type_finalization" returns the type of finalization messages.  
Finalization messages are used by the MPS see signal to the client that an 
object has become finalizable.  In addition to the usual methods applicable to 
messages (see @@@@_blah_), finalization messages support the 
mps_message_finalization_ref method which returns a reference to the registered 
object (see @@@@_blah_ for details).


EXAMPLE

{
  mps_message_type_t type;

  if(mps_message_queue_type(&type, arena)) {
    if(type == mps_message_type_finalization()) {
      process_finalization_message_from_queue();
    } else {
      unknown_message_type();
    }
  }
}


ERROR HANDLING

Cannot fail.
