NAME

mps_message_poll


SUMMARY

"mps_message_poll" is used to determine whether there are currently any 
messages on the message queue of the specified arena.


ASSOCIATED PROTOCOLS

Message.


SYNTAX

mps_bool_t mps_message_poll(mps_arena_t arena)


ARGUMENTS
