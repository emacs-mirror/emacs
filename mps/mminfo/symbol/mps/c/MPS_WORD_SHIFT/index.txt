NAME

MPS_WORD_SHIFT


SUMMARY

"MPS_WORD_SHIFT" is a preprocessor macro defined in "mpstd.h".  It is the 
logarithm in base 2 of MPS_WORD_WIDTH (so 1 << MPS_WORD_SHIFT == 
MPS_WORD_WIDTH).  The value of MPS_WORD_SHIFT is platform-dependent.  Typical 
values are 5 and 6.


ASSOCIATED PROTOCOLS

None.


TYPE

Integral constant.


RESOURCES

mpstd.h.


DESCRIPTION

See summary.


EXAMPLE

#define MPS_WORD_SHIFT 5
