NAME

mps_arena_release


SUMMARY

"mps_arena_release" puts the specified arena into the unclamped state.


ASSOCIATED PROTOCOLS

Arena.


SYNTAX

extern void mps_arena_release(mps_arena_t) ;


ARGUMENTS


INITIAL/DEFAULT VALUES

Not applicable.


RETURNED VALUES

Not applicable.


RESOURCES

Not applicable.


DESCRIPTION

"mps_arena_release" puts the specified arena into the unclamped state.  While 
an arena is unclamped, garbage collection, object motion, and other background 
activity can take place.
