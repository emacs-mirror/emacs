NAME

mps_fmt_create_B


SUMMARY

Function for create a format of variety B.


ASSOCIATED PROTOCOLS

format


SYNTAX

mps_res_t mps_fmt_create_B(mps_fmt_t *fmt_o, mps_arena_t arena, mps_fmt_B_t 
fmt_B);


TYPE


ARGUMENTS

mps_arena_t arena
The arena in which to create the format

mps_fmt_B_t fmt_B
Format description of variety B.  


INITIAL/DEFAULT VALUES



RETURNED VALUES

mps_fmt_t *fmt_o
Returned format that has been newly created.

mps_res_t return
Status result.


RESOURCES


DESCRIPTION

This function creates a format from a user format specification of variety B.  
It is very similar to mps_fmt_create_A.


EXAMPLE

mps_fmt_t create_format(mps_arena_t arena)
{
  mps_fmt_B_s my_format_B = { my_alignment, &my_scan, &my_skip, &my_copy, 
&my_fwd, &my_isfwd, &my_pad, &my_class };
  mps_fmt my_format;
  mps_res_t res;
  
  res = mps_fmt_create_B(&my_format, arena, my_format_B);

  assert(res != MPS_RES_OK);

  return my_format;
}  


ERROR HANDLING

The MPS may exhaust some resource in the course of mps_fmt_create_B and will 
return an appropriate error code in such circumstances.
