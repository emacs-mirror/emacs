NAME

mps_arena_commit_limit


SUMMARY

Returns the current commit limit associated with the arena in bytes.


ASSOCIATED PROTOCOLS

Arena


TYPE

size_t mps_arena_commit_limit(mps_arena_t arena)


ARGUMENTS

arena -- the arena


RETURNED VALUES

Returns the current commit limit as a number of bytes in a size_t


RESOURCES

mps.h


DESCRIPTION

Returns the current commit limit associated with the arena in bytes.  The 
commit limit can be changed using the function mps_commit_limit_set.  The 
commit limit is used to control how much memory the MPS can obtain from the 
OS.  See Arena Protocol for details.


EXAMPLE

limit = mps_arena_commit_limit(arena);


ERROR HANDLING

No errors.


SEE ALSO

mps_arena_committed, mps_arena_commit_limit_set


NOTES

None.
