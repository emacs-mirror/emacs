NAME

mps_assert_default


SUMMARY

"mps_assert_default" returns a pointer to the default assertion handler. It is 
intended to be used in conjunction with "mps_assert_install".


ASSOCIATED PROTOCOLS

Assertion.


SYNTAX


ARGUMENTS

None.


INITIAL/DEFAULT VALUES

None.


RETURNED VALUES

A pointer to the default assertion handler.


EXCEPTIONS


RESOURCES


DESCRIPTION

"mps_assert_default" returns a pointer to the default assertion handler. It is 
intended to be used in conjunction with "mps_assert_install".

You may also call the default assertion handler directly, in which case the 
first, second, and third arguments must be non-NULL pointers to zero-terminated 
strings.
