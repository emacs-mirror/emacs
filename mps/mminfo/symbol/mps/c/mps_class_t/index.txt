NAME

mps_class_t


SUMMARY

"mps_class_t" is the type of pool classes.  It is opaque.


ASSOCIATED PROTOCOLS

None


RESOURCES

mps.h


DESCRIPTION

"mps_class_t" is the abstract type of pool classes.  A pool class may be 
obtained by calling the class function for the appropriate class, such as 
"mps_class_amc" for the AMC class.  A pool class is used when creating a pool 
with "mps_pool_create" or "mps_pool_create_v".
