mps_arena_spare_commit_limit

Retrieves the value of the spare commit limit (previously set with 
mps_arena_spare_commit_limit_set).

Arena.


TYPE

extern size_t mps_arena_spare_commit_limit(mps_arena_t arena)

mps_arena_t arena
  Specifies the arena to retrieve the spare commit limit of.


RETURNED VALUES

Returns, as a size_t, the value of the spare commit limit.

mps.h

Returns the current value of the spare commit limit which is the value most 
recently set with mps_arena_spare_commit_limit_set.  (See 
symbol.mps.c.mps_arena_spare_commit_limit_set for details).
