                           DEVELOPMENT ROLES
                               type.role
                               draft type
                             drj 1995-10-23

Rules: rule.universal

Tag template: role.<name>


Tagging:

Should be tagged with a one-word description of the role.  Eg, role.implementor


Purpose:

A role defines a specific set of goals and responsibilities.  A role is an area 
of expertise on some aspect of the Memory Management System.


Content:

There are fields for goals and responsibilities and duties.
