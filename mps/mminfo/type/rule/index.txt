                                RULE SET
                               type.rule
                               draft type
                           richard 1995-08-09

Rules: rule.rule;rule.universal

Tag template: rule.<doctype>.<name>


Tagging:

Should be tagged rule.<doctype> where doctype is the type of document that the 
the rules apply to.  Each rule in the document should be individually tagged.  
There should a .scope tag describing the scope of the rules.


Purpose:

To assist particular roles during review.  They are not intended to assist 
development (for that see guide.*).


Content:

Rules should contain specific and accurate statements against which a document 
can be checked.
