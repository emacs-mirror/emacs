                       PROGRESS STAGE DEFINITION
                               type.stage
                             obsolete type
                           richard 1995-08-22

Rules: rule.universal

Tag template: stage.<code>


Tagging:

Stages are short adjectives.  Stage documents must be tagged "stage.<code>" 
where <code> is the stage adjective.  For example, "stage.finished".


Purpose:

A document of type "stage" is a stage definition, which defines the meaning of 
a particular stage code.  Stages are used in objectives and issues to indicate 
progress.


Content:

The stage definition document should contain a short description of what it 
means for something to be at this stage, and guidelines on when to use apply 
it.
