PAGE: No rule set shall exceed a single page (60 lines).

EXIT: All rule sets shall have been Inspected against these rules
before being put into practice.

REF: All rule sets shall have a unique reference tag.

GEN: All rule sets shall follow the generic rule set rule.generic.

NEW: Rule tags shall not be re-used for basically different rules.
