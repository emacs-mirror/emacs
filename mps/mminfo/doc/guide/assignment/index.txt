INTRODUCTION

.scope: This document describes conventions for fields in assignments within 
change documents.

.readership: This document in intended for any user of MM Evolution.

.update: See proc.darwin.update.

CONVENTIONS

.desc: The assignment description should give details of what part of the 
change each assignment represents.  It should refer to the .sol.* items from 
the request, e.g. "See request.foo.123456.sol.frob".
