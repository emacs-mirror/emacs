Memory Pool System Project

Memory Management Information Database

Nick Levine, Ravenbrook Limited, 2001-10-08

1. Introduction

This document catalogues the Memory Management Information Database.

The readership of this document is anyone interested in the project.

This document is Ravenbrook company confidential.

2. The Database

attachments/ Flat directory of files attached to one or more documents in the Notes database.
analysis/ Analysis documents.
arch/ Architectures.
bib/ Bibliographies.
book/ Books (in addition to books in the bibliography).

check/ Checklists.

comments/ Comments.
design/ Design documents.
diagram/ Diagrams.

font/ Fonts.
glossary/ Glossary.
goal/ Goal documents.
guide/ Guidelines.

importance/ Importance.
interest/ Stakeholder interest assessments.
inventory/ Inventories.
irc/ Logs of IRC sessions.
issue/ Issues. These predate Darwin (the product evolution database).
kpa/ Key Process Areas.
mail/ Some quotations from mail messages.
map/ Maps.
meeting/ Meetings.
obj/ Objectives of the MM group.
os/ Operating systems.
overview/ Overviews.
paper/ Papers.
part/ Parts.
person/ People.

platform/ Platforms.
proc/ Procedures.
process/ Process documentation.
project/ Projects.
protocol/ Protocols.
report/ Reports.
req/ Requirements.
resource/ Resources.

type/ Project types.
