ABSTRACT:

This book describes Digital's next generation RISC architecture, for which the 
internal code name is Alpha.  References herein to the code name Alpha are 
included for purposes of continuity.

This book is directly derived from sections of the Alpha System Reference 
Manual and is an accurate representation of the described parts of the Alpha 
architecture.
